# node
[NPM安装](http://n.thepana.com/2013/09/21/nodejs_install/)
## 从源码安装
```bash
    ##下载源码
    git clone https://github.com/nodejs/node
    ## make 安装
    ./configure
    make
    sudo make install
```
## 使用N来管理Node
```bash
    git clone https://github.com/tj/n.git
    sudo make
    n latest
```
